I built this portfolio from scratch.

I wanted a way to showcase both my instructional design skills and my front-end development skills. So instead of using a template, I created this website from scratch using HTML, CSS, & JavaScript.

My Process.

This was no easy task! Here's how I built this, step by step.

  1. Sit down and think about the goal of the portfolio
  2. Write an outline & gather content
  3. Create a mockup in Adobe XD
  4. Build out the frame and styling with HTML & CSS
  5. Plan and implement the functionality using JavaScript
  6. Review & Revise
  7. Publish and host the site!
