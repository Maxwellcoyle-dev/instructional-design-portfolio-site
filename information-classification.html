Information Classification

Overview

The course is a short and effective software training that helped drive product adoption. Shorter micro-learnings not only make learning more approachable but can also serve as a resource for employees going forward.

Design Insights

Who were the learners?

Employees at a large pharmaceutical company who were asked to use Microsoft Information Protection to apply data security classification labels to new documents.

What were the learning objectives?

After completing this training, learners will be able to:

  1. Access the Data Classification labels when working in Microsoft office
  2. Accurately assign the appropriate data security classification label to Microsoft Office documents using Microsoft Information Protection

How were the learning objectives met?

This training effectively demonstrated how to access the Data Classification labels from within Microsoft Office applications. Additionally, the practice exercise gave learners experience in determining the appropriate classification label for different types of documents.
